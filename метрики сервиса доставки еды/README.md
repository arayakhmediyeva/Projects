# Аналитика бизнес-метрик сервиса доставки еды
## О проекте
Проект посвящён анализу ключевых показателей сервиса доставки еды в Саранске за май–июнь 2021 года.
Цель — выявить закономерности в поведении пользователей, оценить конверсию, удержание клиентов и LTV по ресторанам и блюдам.
## Использованные инструменты
- SQL — выборка и агрегация данных
- DataLens — визуализация и построение интерактивных графиков
- CSV — хранение и импорт исходных данных
## Дашборд
Ключевые метрики:
- Уникальные активные пользователи (MAU)
- Конверсия и её динамика по дням
- Средний чек и его изменения во времени
- Коэффициент удержания клиентов (Retention Rate)
- LTV по ресторанам
- LTV по блюдам

Ознакомиться с интерактивным дашбордом можно по ссылке: [https://datalens.yandex/ukw3w5qatbmmf]

## Основные результаты
Средняя конверсия — 43%. Средний чек — 135,9 ₽. Уровень удержания клиентов к 7-му дню — 4–5%. Топовые рестораны по LTV: Гурманское Наслаждение и Гастрономический Шторм. Высокое LTV отмечено для блюд: запечённая брокколи с травами и говяжьи шашлыки в песто из кинзы.

## Выводы

Пользовательская активность растёт ближе к концу месяца.
Женская аудитория демонстрирует более низкое удержание, чем мужская.
Наибольший вклад в прибыль приносят несколько ключевых ресторанов, что указывает на концентрацию заказов.
Анализ данных помогает корректировать маркетинговые кампании и программы лояльности.
