# Аналитика рынка недвижимости

## О проекте  
Проект посвящён анализу рынка недвижимости с использованием данных о квартирах: количество комнат, площадь, цена, дата объявления и локация. Цель — выявить тенденции на рынке для оптимизации работы агентства недвижимости.  

## Использованные инструменты  
- `SQL` — обработка и выборка данных  
- `DataLens` — построение интерактивных дашбордов  
- `CSV` — формат хранения и импорта данных
  
## Дашборд
Дашборд состоит из двух аналитических листов:
1. **Аналитика рынка недвижимости** - Ключевые показатели: количество квартир, средняя площадь, среднее число комнат, динамика предложений и цен, ТОП-локации.
2. **Детализация рынка** - Средняя стоимость квадратного метра по городам, структура предложений по типу жилья (апартаменты / не апартаменты), планировке (открытая / закрытая), наличию парков и водоёмов поблизости.

Ознакомиться с интерактивным дашбордом можно по следующей [ссылке](https://datalens.yandex/nyaf7rtlncly9).

## Основные результаты  
- Уникальных квартир: 23 650  
- Среднее количество комнат: 2  
- Средняя площадь: 60 м²  
- Динамика роста предложений и стабилизация цен  
- ТОП-локации: Санкт-Петербург, Раппино, Мурино и др.

## Выводы  
Рынок недвижимости растёт, что подтверждается увеличением предложений и сокращением времени продажи. Данные помогают оптимизировать ценообразование и выбирать перспективные регионы для работы агентства.
