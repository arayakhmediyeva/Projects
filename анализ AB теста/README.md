# Разработка A/B-тестирования и анализ результатов
**Описание проекта**
Для компании, разрабатывающей развлекательное приложение с «бесконечной» лентой контента (short-video style) необходимо рассчитать параметры A/B-теста, который позволит проверить гипотезу о влиянии нового алгоритма рекомендаций, и проанализировать его результаты.

**Цель проекта** - оценить влияние нового алгоритма рекомендаций на ключевые продуктовые метрики, проверить статистическую значимость изменений.

**Задачи:**
- спроектировать A/B-тест для проверки гипотезы о том, что новая модель рекомендаций повышает интерес пользователей,
- рассчитать параметры теста (размер выборки, длительность),
- проверить корректность проведения теста и проанализировать результаты.

**Используемые библиотеки**
- pandas, numpy — анализ и подготовка данных;
- matplotlib, seaborn, plotly — визуализация;
- scipy.stats, statsmodels — статистическая проверка гипотез;
- Jupyter Notebook — проведение анализа и подготовка отчёта.

A/B тест был проведен на основе гипотезы о том, что доля успешных сессий при внедрении нового алгоритма вырастет.

Необходимый размер выборки для каждой группы составил 41040. Рассчитанная длительность A/B-теста при текущем уровне трафика в 9907.36 пользователей в день составил 9 дней.

**Вывод**
По результатам A/B-эксперимента нулевая гипотеза (H₀) о том, что доля успешных сессий не изменилась, не нашла подтверждения. Следовательно, внедрение нового алгоритма рекомендаций привело к росту доли успешных сессий.
