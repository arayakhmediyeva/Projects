# Ключевые бизнес-метрики фонда АиФ «Доброе сердце»
## О проекте
Интерактивный дашборд создан для анализа ключевых показателей благотворительного фонда АиФ. Доброе сердце.
Он помогает отслеживать активность пользователей, удержание благотворителей и финансовые метрики по регионам и когортам.

**Цель проекта** - предоставить инструмент для оценки эффективности работы фонда: анализировать активность благотворителей (MAU); оценивать Retention Rate по полу и времени участия; измерять средний чек и жизненную ценность доноров (LTV); выявлять различия по городам и когортам.

## Основные метрики

- MAU: от 353 до 1 895 пользователей в месяц
- Retention Rate:
Мужчины — до 0.38 после 17 месяцев;
Женщины — удержание стабилизируется около 0.14
- Средний чек:
Мужчины — 890 ₽;
Женщины — 629 ₽
- Наибольший LTV:
Екатеринбург — 4 737 ₽;
Москва — 2 996 ₽;
Санкт-Петербург — 1 456 ₽.

## Ключевые визуализации

- Динамика MAU — изменение месячной активности пользователей за 2021–2024 гг.
- Retention Rate по полу — удержание благотворителей по когортам.
- Средний чек по когортам — сравнение среднего размера пожертвований у мужчин и женщин.
- LTV крупных городов — жизненная ценность доноров по основным регионам России.

## Используемые инструменты
- Yandex DataLens — визуализация и аналитика
- SQL — подготовка и обработка данных
- CSV — хранение и импорт данных
## Выводы
Активность пользователей фонда растёт волнообразно, достигая пиков в периоды крупных кампаний.
Мужчины демонстрируют более высокий уровень удержания и больший средний чек.
Самый высокий LTV наблюдается у доноров из Екатеринбурга.
Полученные данные позволяют оптимизировать маркетинг и стратегию взаимодействия с аудиториями.
