# Анализ данных бронирования билетов на сервисе Яндекс.Афиша
## Описание проекта
В проекте проводится исследовательский анализ данных о бронировании билетов на сервисе Яндекс Афиша за период с 1 июня по 30 октября 2024 года.

**Цель исследования** — выявить ключевые закономерности в поведении пользователей, определить популярные события и билетных партнёров, а также провести статистический анализ различий в активности пользователей мобильных и десктопных устройств.

**Задачи проекта:**
- Предобработка данных и подготовка их к исследованию;
- Анализ распределения заказов по сегментам и их сезонные изменения;
- Анализ осенней активность пользователей;
- Выявление популярных событий и билетных партнёров;
- Статистический анализ данных и проверка гипотез;
- Формулировка вывода и рекомендаций.

## Описание данных
Данные разделены на два датасета final_tickets_orders_df.csv и final_tickets_events_df.csv. 
1. **final_tickets_orders_df.csv** 
Датасет включает информацию обо всех заказах билетов, совершённых с двух типов устройств — мобильных и стационарных.
2. **final_tickets_events_df.csv** 
Датасет содержит информацию о событиях, представленных на платформе.

## Используемые библиотеки
Python
- Pandas — анализ и обработка данных
- Matplotlib / Seaborn / Plotly — визуализация данных
- NumPy — статистические расчёты
- SciPy / Statsmodels — проверка статистических гипотез
- Jupyter Notebook — аналитические исследования

## Итоговый вывод
Результаты анализа помогут понять поведение пользователей Яндекс Афиши, оптимизировать работу сервиса, а также повысить эффективность привлечения и удержания клиентов на разных платформах (мобильной и десктопной).
