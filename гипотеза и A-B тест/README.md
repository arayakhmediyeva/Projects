# Аналитический проект: проверка гипотезы о лояльности пользователей и анализ результатов A/B-теста
## Описание проекта
Проект состоит из двух аналитических частей, объединённых общей целью — исследовать поведение пользователей и оценить эффективность изменений в цифровых продуктах.
- Часть 1 — проверка гипотезы о различии в лояльности пользователей сервиса Яндекс Книги между Москвой и Санкт-Петербургом.
- Часть 2 — анализ результатов A/B-теста редизайна сайта интернет-магазина BitMotion Kit.
## Часть 1. Проверка гипотезы о лояльности пользователей сервиса Яндекс Книги
**Цель исследования** - проверить, существует ли статистически значимое различие в средней лояльности пользователей (в часах активности) между жителями Москвы и Санкт-Петербурга.

**Метрика:** количество часов активности пользователя (hours)

**Группы:** пользователи из Москвы и Санкт-Петербурга

**Тип теста:** односторонний t-тест Стьюдента;

**Вывод**
Разница в средней лояльности между пользователями Москвы и Санкт-Петербурга не является статистически значимой.
Это может говорить о схожем уровне вовлечённости пользователей в обоих городах.
Вероятно, на лояльность влияют не географические различия, а другие факторы — жанровые предпочтения, рекомендации сервиса или время использования приложения.
## Часть 2. Анализ результатов A/B-теста интернет-магазина BitMotion Kit

**Цель анализа** - проверить, улучшает ли новая версия сайта конверсию в покупку по сравнению с текущей версией.

**Задачи**
- Проверить корректность проведения эксперимента
- Провести исследовательский анализ данных (EDA)
- Провести статистический анализ
- Сделать выводы и рекомендации

**Используемые библиотеки**
- pandas, numpy — анализ и подготовка данных;
- matplotlib, seaborn, plotly — визуализация;
- scipy.stats, statsmodels — статистическая проверка гипотез;
- Jupyter Notebook — проведение анализа и подготовка отчёта.
## Итог проекта
- Часть 1 — проверена гипотеза о различии лояльности пользователей Яндекс Книги из Москвы и Санкт-Петербурга.
→ различий не выявлено, пользователи демонстрируют схожую вовлечённость.
- Часть 2 — проведена оценка результатов A/B-теста нового интерфейса BitMotion Kit.
→ определены закономерности поведения пользователей и эффективность новой версии сайта.
